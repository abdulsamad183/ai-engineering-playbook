# Evaluation Templates

> Harnesses for prompt, RAG, agent, and model comparison evaluation.

---

## Modules

| File | Purpose |
|------|---------|
| `prompt_eval.py` | Must-include string checks |
| `rag_eval.py` | Faithfulness proxies |
| `agent_eval.py` | Success + efficiency |
| `model_comparison.py` | Latency and cost benchmarks |

---

## Usage

Import modules into CI regression suites. Pair with [AI Evaluation handbook](../../../domains/ai-evaluation/README.md).
