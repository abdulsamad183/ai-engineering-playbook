## Creating a New Technology Index

1. Create `{technology-name}.md` in this folder when 2+ documents reference the technology.
2. Include: overview, relevant documents by domain, configuration references, and comparison links.
3. Add an entry to this README.
4. Register the technology tag in the [tag registry](../tags/tag-registry.md).
