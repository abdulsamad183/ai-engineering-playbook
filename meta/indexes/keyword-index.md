# Keyword Index

> Alphabetical index of keywords extracted from document front matter.
> Keywords enable natural-language discovery beyond structured tags.

---

## How Keywords Work

- Keywords are listed in each document's YAML front matter.
- They supplement tags with free-form, natural-language terms.
- Max 10 keywords per document.
- This index is updated when documents are published.
