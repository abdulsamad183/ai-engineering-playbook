# MCP Resource Design Checklist

- [ ] Stable hierarchical URI scheme
- [ ] MIME type in metadata
- [ ] ACL filters `resources/list` and `resources/read`
- [ ] Pagination for large catalogs
- [ ] Version or etag for cache invalidation
- [ ] No path traversal in dynamic URIs
- [ ] Size limits on read payloads

See [MCP Resources](../domains/mcp/mcp-resources.md).
