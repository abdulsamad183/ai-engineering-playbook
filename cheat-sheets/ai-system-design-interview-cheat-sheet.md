# AI System Design Interview Cheat Sheet

1. Clarify functional + non-functional requirements
2. Estimate QPS, storage, tokens/day
3. Draw: client → API → orchestration → LLM/RAG/agents → data
4. Deep dive: context, streaming, failure modes
5. Monitoring, cost, tradeoffs
