# Evaluation Workflow Cheat Sheet

1. Define success criteria (quality + latency + cost)
2. Build/version golden dataset
3. Instrument system (versions, traces)
4. Run automated metrics (+ human on failures)
5. Gate deploy in CI
6. Monitor online with sampling
7. Iterate — add cases per incident

See [Evaluation Architecture](../domains/ai-evaluation/evaluation-architecture.md).
