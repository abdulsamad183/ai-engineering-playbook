# Agent Production Deployment Checklist

- [ ] Max steps / cost / duration enforced
- [ ] Checkpointing for long runs
- [ ] Queue-backed workers
- [ ] Traces per step (OpenTelemetry)
- [ ] HITL on destructive tools
- [ ] Per-tenant credentials
- [ ] Eval suite in CI
- [ ] DLQ for failed runs

See [Production Agent Engineering](../domains/ai-agents/production-agent-engineering.md).
