# Cheat Sheets

> Quick reference cards for technologies, patterns, and workflows.

---

## Purpose

Cheat sheets provide at-a-glance reference for topics you use frequently. They complement detailed domain documents — link to the full guide for depth.

---

## Standards

Cheat sheets follow the [Cheat Sheet template](../meta/templates/cheat-sheet.md):

- Essential commands and code patterns
- Key concepts in table format
- Quick troubleshooting fixes
- Do's and don'ts
- Links to full documentation

3. Keep it to one screen of content when possible.
4. Link to the full domain guide.
