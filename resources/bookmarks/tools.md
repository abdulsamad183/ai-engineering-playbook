# Tools

> Curated AI engineering tools and services.

- [OpenAI Platform](https://platform.openai.com/) — GPT APIs, Assistants, embeddings, and usage dashboards.
- [Anthropic Console](https://console.anthropic.com/) — Claude models, workbench, and API keys.
- [Google AI Studio](https://aistudio.google.com/) — Gemini prototyping and API key management.
- [GroqCloud](https://console.groq.com/) — Fast inference for open models.
- [OpenRouter](https://openrouter.ai/) — Unified API across many LLM providers.
- [Ollama](https://ollama.com/) — Run local LLMs on your machine.
- [Qdrant](https://qdrant.tech/) — Vector database for similarity search and filtering.
- [Pinecone](https://www.pinecone.io/) — Managed vector database.
- [LangSmith](https://www.langchain.com/langsmith) — Tracing, evals, and prompt debugging for LLM apps.
- [Langfuse](https://langfuse.com/) — Open-source LLM observability and evals.
- [Weights & Biases](https://wandb.ai/) — Experiment tracking and model monitoring.
- [Hugging Face](https://huggingface.co/) — Models, datasets, Spaces, and Inference Endpoints.
