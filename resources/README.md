# Resources

> Curated external resources, tools, and references.

---

## Organization

```
resources/
├── bookmarks/         # Categorized link collections
└── README.md
```

---

## Bookmark Categories

Organize bookmarks by topic in `bookmarks/`:

| File | Content |
|------|---------|
| `tools.md` | AI engineering tools and services |
| `documentation.md` | Official docs and API references |
| `blogs.md` | Engineering blogs and newsletters |
| `courses.md` | Courses and learning materials |
| `communities.md` | Forums, Discord, Slack communities |
| `papers.md` | Key research papers (links to summaries in domains/papers/) |

---

## Adding Resources

1. Add links to the appropriate bookmark file.
2. Include a one-line description for each link.
3. Prefer official documentation over third-party summaries.
4. Remove dead links during periodic reviews.

### Format

```markdown
- [Resource Name](https://url) — One-line description of why it's useful.
```
