# Phoenix Evaluation

## Overview

**Arize Phoenix** combines LLM observability with evaluation and embedding analysis.

## Strengths

- Trace visualization
- Embedding drift and cluster analysis
- RAG relevance debugging
- Open-source core

## Limitations

- Requires instrumentation setup
- Full platform features need Arize cloud

## Ideal Use Cases

- Debug retrieval quality visually
- Monitor embedding drift after index updates
- Root-cause RAG failures from traces
