# AutoGen for Agents

## Overview

**AutoGen** enables **multi-agent conversations** — agents message each other with optional code execution.

| Aspect | Detail |
|--------|--------|
| **Strengths** | Flexible group chat patterns, research prototypes |
| **Weaknesses** | Conversation can diverge; cost control needed |
| **Production** | Requires strict termination and tool sandbox |
| **Best for** | Debate, coding with user proxy |
