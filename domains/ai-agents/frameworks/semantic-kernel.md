# Semantic Kernel for Agents

## Overview

**Semantic Kernel** provides **plugins** (tools), planners, and memory abstractions for enterprise .NET/Python stacks.

| Aspect | Detail |
|--------|--------|
| **Strengths** | Enterprise integration, Azure alignment |
| **Weaknesses** | Heavier model than minimal Python frameworks |
| **Best for** | Microsoft ecosystem, plugin-heavy enterprise apps |
